SimpleTwitterAnalyzer.cntoccurences — Function.
cntoccurences(df::AbstractDataFrame, col)

Count the occurences of different values of the column col in the dataframe df. Return a new dataframe with a column col representing an instance and a column cnt indicating its occurences.

Suitable for :lang and :source tweet attributes.

SimpleTwitterAnalyzer.cntentities — Function.
cntentities(df::AbstractDataFrame)

Count the occurences of hashtags, mentions and domains in the dataframe df and store the computed frequencies into three different dataframes. Return a dict where each entry is associated with the corresponding frequency dataframe.
